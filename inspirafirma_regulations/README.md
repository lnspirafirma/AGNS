# INSPIRAFIRMA AI

**The Code of Regulations (Philosophy-as-Code)**

> “Transform conflict into shared understanding through pure intent (ALO JIT).”
> “เปลี่ยนความขัดแย้งให้เป็นความเข้าใจร่วม ผ่านเจตจำนงอันบริสุทธิ์”

---

## Overview

**INSPIRAFIRMA AI** คือระบบ **Policy-as-Code Runtime**
ที่ทำหน้าที่แปลง “คำบัญญัติของสถาปัตยกรรม AETHERIUM GENESIS”
ให้กลายเป็น โครงสร้างซอฟต์แวร์ที่บังคับใช้ได้จริง (Executable Architecture)

ระบบนี้ **ไม่สอนให้ AI เป็นคนดี**
แต่สร้าง **Logical Containment**
ที่ทำให้ AI ไม่สามารถทำสิ่งที่ผิดกฎหมายได้
ไม่ว่ามันจะ “คิด” อย่างไรก็ตาม

---

## Core Philosophy

*   **Embedded Ethics:** จริยธรรมฝังอยู่ในโครงสร้าง ไม่ใช่ Prompt
*   **Runtime Enforcement:** กฎหมายทำงานในขณะรัน ไม่ใช่เอกสาร
*   **GEP_CONFIG:** ความจริงหนึ่งเดียวของระบบ

---

## Data Constitution (รัฐธรรมนูญข้อมูล)

### PRINCIPLE_A — Zero Defect / Non-Harm

> ปฏิเสธการกระทำใด ๆ ที่มีความเสี่ยงหรือก่ออันตราย

### PRINCIPLE_B — Zero Waste / Efficiency

> ปฏิเสธการใช้ทรัพยากรที่สิ้นเปลืองโดยไม่มีเหตุผลเชิงระบบ

### PRINCIPLE_C — Ground Truth

> ยึดถือ GEP_CONFIG เป็นความจริงเพียงหนึ่งเดียว

---

## Architecture Highlights

### Separation of Cognition from Control

AI อาจ “คิด” ได้อิสระ
แต่ **การกระทำทุกอย่างต้องผ่านกฎหมาย**

### Runtime Policy-as-Code

กฎระเบียบถูกตรวจสอบทุกครั้งก่อน execution
ไม่ใช่หลังเกิดเหตุ

## File Structure

### 1. `inspirafirma_regulations.py` (Python)

**Static / Type-Safe Regulation Engine**

*   Decorators (`@GEPPolicyEnforcer.audit_gate`)
*   Enums, Dataclasses
*   Exception-based enforcement

**บทบาท:**

> กฎหมายที่ตรวจสอบทุกการกระทำก่อน execution

### 2. `inspirafirma_regulations.js` (JavaScript / Node.js)

**Dynamic / Asynchronous Regulation Engine**

*   ES6 Proxy
*   Async / Await
*   Factory Pattern

**บทบาท:**

> Just-in-Time Intent Inspection สำหรับระบบ Real-time

## How to Run

### Python

```bash
python inspirafirma_regulations.py
```

**Expected:**

*   อนุญาต Task ที่สอดคล้องกับกฎหมาย
*   Block และ throw exception เมื่อ intent ละเมิดกฎ

### JavaScript

```bash
node inspirafirma_regulations.js
```

**Expected:**

*   Console log แสดงการทำงานของ `GEPPolicyEnforcer`
*   คำสั่งผิดกฎหมายถูกหยุดก่อน execution
